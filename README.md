# Employee Management System Using MERN Stack

## Overview

The Employee Management System is a full-stack MERN based web application developed using MERN stack to streamline employee related administrative tasks in an organization.The system provides secure role-based access for admin and employees allowing seamless and efficient management of employee records, departments, leave application and approval and salary details.

The project has been built to demonstrate implementation of full-stack web development concepts such as authentication , CRUD operations, REST Api integration , role-based access control and MongoDB Management in real-world scenario. 

## Features

1. Admin and Employee login system
2. User authentication and authorization 
3. Role Based Access Control (RBAC)
4. Admin Dashboard
5. Employee Dashboard
6. Employee Record  Management
7. Department Management
8. Leave Application and Approval
9. Salary Management
10. Form Validation
11. Error handling
11. REST API integration
12. Secure Database Connectivity (MongoDB)

## Tech Stack
1. Frontend : React.js (Frontend JavaScript Framework)
             Axios 

2. Backend: Express.js (API handling)
           Node.js (Server side Javascript runtime environment)
           
3. Database: MongoDB

4. Tools:
a. Bcrypt (password hashing)
b. CORS  (Browser based security mechanism)
c. JWT  (Security and tokenization)
d. Mongoose (ODM library for MongoDB)
e. Multer (middleware handling multi-part/form-data)
f. Nodemon (Server side node wrapper)
g. Path (handling file path) 

5. Apparatus: VS Code


## Modules:

## Installation and Setup

### Clone the repository
```bash
git clone <your-github-repo-link>
cd employee-management-system
```

### Setup backend
```bash
cd backend
npm install
npm start
```

### Setup frontend
```bash
cd frontend
npm install
npm run dev
```

## Workflow
1. Admin logs into the system.
2. Manages departments, employee records , leave approval and salary details.
3. Employees log innto the sytem.
4. Accesses personal dashboard , tracks salary , views profile and apply leaves.
5. Data is stored and retrieved using MongoDB through REST APIs.
